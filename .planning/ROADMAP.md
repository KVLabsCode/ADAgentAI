## Phase 1: Shared Components

**Goal:** Build the Linear-style design foundation that all pages will use

**Requirements:**
- SHARED-01: Navigation bar
- SHARED-02: Footer
- SHARED-03: Dark theme (gradients, glows)
- SHARED-04: Scroll animations
- SHARED-05: Hover states / micro-interactions
- SHARED-06: Typography system

**Success Criteria:**
1. Nav bar renders with logo, links, and styled CTA button
2. Footer renders with multi-column link organization
3. Dark theme variables defined with gradient/glow utilities
4. CSS scroll reveal animations work
5. Buttons and links have smooth hover transitions
6. Typography scale matches Linear's hierarchy

**Dependencies:** None (foundation phase)

## Phase 2: Blog Listing

**Goal:** Redesign blog index page with Linear-style card grid

**Requirements:**
- BLOG-01: Category tabs/filters
- BLOG-02: Search functionality
- BLOG-03: Responsive card grid
- BLOG-04: Blog card component

**Success Criteria:**
1. Category tabs filter posts by category
2. Search bar filters posts by title/content
3. Cards display in 2-3 column responsive grid
